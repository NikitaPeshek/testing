# консольный git команды
git init - создание репозитория
git status - проверка временношо хранилища что там естьи что есть к выгрузке или нет
git add . - отправка всех фалов за исключение игнорированных в временное хранилище
git add *.html - отправка всех файлов с расширением htlm в временное хранилище
git rm {название файла} - удаляет данный файл из времемнного хранилища
git commit - отправка в внешние репозиторий 
git push - отправляет локальный репозиторий на внешний
git pull - получить последние изменения с внешнего на локальный
git clone - копирует внешний репоз в текущюю папку
